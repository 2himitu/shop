# 홈쇼핑 사이트

- 기능은 로그인시 이용 가능하다.
- 아이디,비밀번호 찾기가 가능하다.
- 상품의 조회가능가 하다.
- 장바구니의 저징시 수량,색상의 선택이 가능하다.
- 비밀번호 찾기시 가입시 입력한 메일에 임시 비밀번호를 전송한다.
- 장바구니에 원하는 물품을 저장 가능하다

- 관리자의 계정으로는 유저리스트를 통해 일반유저의 계정 삭제가 가능하다.
